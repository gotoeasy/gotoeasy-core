# `gotoeasy-core`
GotoEasy系列的核心包，目的为提供常用功能方便使用。

- depend on & thanks to `slf4j` http://github.com/qos-ch/slf4j

## 主要内容
```
gotoeasy-core/
├── bus （总线）
├── config （Js配置文件）
├── converter （数据类型转换）
├── reflect （反射类）
└── util （工具类）
```
