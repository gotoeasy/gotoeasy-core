# `gotoeasy-core`
GotoEasy系列的核心包，没有第三方包的依赖，目的为提供常用功能方便使用。

Maven使用
```xml
<dependency>
    <groupId>top.gotoeasy</groupId>
    <artifactId>gotoeasy-core</artifactId>
    <version>x.y.z</version>
</dependency>
```
Gradle使用
```gradle
compile group: 'top.gotoeasy', name: 'gotoeasy-core', version: 'x.y.z'
```

## 主要内容
```
gotoeasy-core/
├── bus （总线）
├── compiler （动态编译）
├── config （Js配置文件）
├── converter （数据类型转换）
├── log （日志系统动态选择）
├── reflect （反射类）
└── util （工具类）
```
